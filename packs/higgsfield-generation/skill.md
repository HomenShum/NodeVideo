# Higgsfield generation

Use the official Higgsfield CLI as the normal provider boundary. Discover models and estimate cost before creating a job. Persist the provider job ID before polling. Download, probe, hash, rights-review, and receipt every accepted output.

The browser adapter is allowed only when an authenticated entitlement is proven to be browser-only. Never assume that a web promotion applies to CLI or MCP. Never upload private, confidential, or third-party media without explicit authorization.

Generated output may support NodeVideo productions, but must not be used to train or improve a competing model without separate legal clearance.
