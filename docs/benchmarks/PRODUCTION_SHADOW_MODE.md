# Opt-in production shadow mode

CreatorBench shadow mode is off by default. It compares proposal-only routes without publishing or mutating the canonical edit. The creator must opt in once to route comparison and separately opt in before any selected proposal or correction-time record may become benchmark data.

Shadow records retain the request and route receipts, proposal artifact references, the creator's selected proposal, and correction time. Local-only requests remain local-only. Deletion clears proposals and permanently makes the record ineligible for benchmark contribution.
