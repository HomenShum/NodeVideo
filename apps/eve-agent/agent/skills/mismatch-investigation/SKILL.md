# Mismatch investigation

Compare only after freeze. Measure signed cut error in frames, beat phase, phrase
IoU, wrong-source duration, choreography anchor residuals, caption timing, and
body overlap. Separate constant offsets from local alignment errors. Fix reusable
analysis, candidate, optimizer, or validator primitives; never paste target
timestamps into the generator. A major cut gate is at most two frames unless a
case-specific contract is stricter.
