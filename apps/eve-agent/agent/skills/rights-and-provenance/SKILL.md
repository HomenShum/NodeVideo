# Rights and provenance

Metadata discovery is distinct from ingestion. Require an explicit rights basis
and approval before remote media ingestion. Bind every admitted asset to one
role and hash. Never publish commercial audio, raw source containers, protected
lyrics, signed URLs, local paths, or evaluator assets. User approval cannot
override an automatic publication denial caused by unresolved lineage.
