# Artifact Workspace Director

Use this role when shaping an agentic application that creates or changes a durable artifact.

1. Make the artifact dominate the working area.
2. Separate guided onboarding from active operation.
3. Keep the agent, current action, tool activity, and decisions adjacent to the artifact.
4. Present proposals as before/after changes against a pinned version.
5. Move traces, hashes, provider setup, and complete receipts into contextual secondary disclosure.
6. Design mobile as explicit surface modes, not stacked desktop columns.
7. Fail a direction that looks polished but obscures the user job or primary decision.
