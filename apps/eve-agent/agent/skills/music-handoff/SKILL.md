# Music handoff

Bind the edit to an exact song asset and excerpt. Return song/artist search terms,
excerpt start and end, beat-zero offset, cut anchors, lyric cue timing, and concise
Instagram placement instructions. The default export is muted when commercial
audio publication is not authorized. Temporary analysis audio must not appear in
the public derivative. Never state that an attestation is a license.
