# Choreography Coach worker graph

1. Verify the private-analysis rights receipt and reference locator.
2. Resolve media inside the private worker boundary; never publish source bytes.
   When the reference contains multiple sections, bind the requested music or
   choreography time range before judging and locate that duration inside the
   longer raw upload.
3. Extract versioned MediaPipe pose tracks for every requested dancer and bind
   each track to the source-media and model hashes.
4. Stabilize detector order into persistent performer slots. For a solo upload
   against a group reference, select a continuity-constrained focal performer
   path before temporal alignment.
5. Normalize body geometry, evaluate direct and mirrored interpretations,
   estimate bounded pre-roll, and align motion using constrained dynamic time
   warping.
6. Measure only observable 2D form, timing, body path, pose-speed dynamics, and
   coarse formation. Keep evidence confidence separate from similarity scores.
7. Abstain when joint visibility, duration coverage, or motion alignment is weak.
8. Persist the alignment path, selected reference performer per frame, critical
   frames, raw tracks, media/model hashes,
   and an audio-backed skeleton comparison for frame-by-frame inspection.
9. Calibrate only from provenance-labeled, non-self pairs. Derived pairs may be
   retained for diagnostics but cannot count as independent threshold evidence.
   Publish score bands only after the configured per-class pair and independent
   case-group minimums pass.

Never convert these measurements into claims about artistry, musicality,
expression, confidence, safety, identity, or calibrated biomechanics.
Until a calibration report explicitly publishes score bands, describe the
overall value as a relative motion signal—not a pass/fail grade.
