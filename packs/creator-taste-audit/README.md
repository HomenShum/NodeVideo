# Creator Taste Audit

This content-neutral capability converts repeatable production forensics into an evidence-bound creator profile and reusable fidelity gates. It supports dance, talking-head, tutorial, comedy, montage, and other short-form formats.

The profile is a prior, not a clone recipe. Case-specific movements, dialogue, products, and timing still come from the new source material. Low-support traits remain provisional, and transfer between formats must be revalidated.
