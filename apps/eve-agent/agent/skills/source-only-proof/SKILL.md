# Source-only proof

1. Admit role-bound asset IDs and SHA-256 hashes. Reject duplicate IDs or hashes.
2. Keep the held-out target outside every generation schema, prompt, mount, credential, trace, and artifact.
3. Prepare and approve the exact source-only proposal digest.
4. Generate analysis, candidates, captions, edit plan, render, and read log.
5. Freeze their immutable hashes while `targetAccess=denied` and evaluator credentials are absent.
6. Only the `proof_critic` may receive the freeze plus the separately admitted target.
7. Report target comparison as evaluation evidence, never as generation input.

Passing means blindness is enforced by access boundaries, not merely claimed in prose.
